==== Windows 平台 ====
  * [[https://code.google.com/p/msysgit/ | mysysgit ]] win平台下的命令行工具
  * [[http://code.google.com/p/gitextensions/ | gitextensions]]可以和win资源管理器集成，与sysgit集成，图形化git命令。  
  * [[http://sourcetreeapp.com/ |SourceTree ]]   SourceTree 是 Windows 和Mac OS X 下免费的 Git 和 Hg 客户端。 
  * [[http://www.syntevo.com/smartgithg/ | SmartGit ]] syntevo 公司的Git和HG的客户端工具，非商业用户免费。。
  * [[http://code.google.com/p/tortoisegit/ | TortoiseGIT ]]  著名的TortoiseSVN的免费git版本。
  * [[http://www.collab.net/giteyeapp| GitEye ]] CollabNet 公司的免费工具。
  * [[http://windows.github.com | GitHubWindws]] Github的win平台的免费客户端工具。

==== Mac 平台====
  * [[http://www.git-tower.com/| Tower ]] mac 下的git客户度，商业工具。
